## Features

<CardGrid>
  <Card title="List view for your models" icon="list-format">
    - server side filters
    - server side pagination
  </Card>

   <Card title="CRUD functionnality" icon="pen">
    - Delete from list view
    - Dedicated create and update pages
  </Card>

  <Card title="Many data types built in" icon="seti:odata">
    Files, images, bitsets, enums, dates, Lucid model relations, Lucid computed fields...
  </Card>

  <Card title="Export functionnality" icon="document">
    - server side filters
    - support for excel, csv, json
  </Card>

  <Card title="Sidebar" icon="seti:image">
    - icons from [tabler icons](https://tabler.io/icons)
    - folders and subfolders (with no limit)
    - easily add links for your custom views
  </Card>

  <Card title="Good DX" icon="seti:typescript">
    - Excellent type inference where you need it
    - JSDoc anotations in every config types to help you find what you need
  </Card>

  <Card title="Easy to use Charts" icon="gitter">
    - common chart types supported
    - dead simple configuration
    - [helpers to fetch common chart data in 1 LoC](/adomin/reference/views/stats/#data-fetcher-helpers)
  </Card>

  <Card title="Get started" icon="open-book">
    - [Installation](/adomin/guides/installation)
    - [Configuration](/adomin/guides/configuration)
  </Card>
</CardGrid>
